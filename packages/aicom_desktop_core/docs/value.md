# aicom_desktop_core — Value

## English

Shared themes, wallet bar, languages, and backup for every AICOM desktop app — consistent UX and one place to fix economics/settings for all eight products.

Общие темы, панель кошелька, языки и бэкап для каждого desktop-приложения AICOM — единый UX и одно место, где чинить economics/настройки для всех восьми продуктов.
